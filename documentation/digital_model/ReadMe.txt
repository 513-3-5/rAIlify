***Example of modern 'Tool (Chain)' for Basel Hack***

This repository contains a fictive example of a new tool plan. Demonstrates the link between a digital machine-readable file (XML) and a graphical topology plan.

***Overview***
Includes the following components:
ExamplePlan.xml
	-A digital machine-readable file containing the data for the topology.
ExamplePlan.pdf
	-The resulting graphical representation of the topology based on the XML input.
